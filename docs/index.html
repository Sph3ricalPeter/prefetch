Features

Everything you need,
nothing you don't

Built for developers who want visual clarity without giving up speed or control.

Commit graph

Canvas-rendered, virtualized DAG. Handles repos with 100k+ commits without breaking a sweat. See branches, merges, and history at a glance.

Git profiles

Auto-switch identity by repo path. Work vs. personal, zero friction.

Inline diffs

Grouped hunks, clear additions and deletions. Click a file, see the change.

GitHub & GitLab

PR badges on branches, PAT stored in your OS keychain, click to open in browser. Works with GitHub and GitLab out of the box.

Undo anything

Ctrl+Z backed by reflog. Even works for ops run in your terminal.

Git LFS

Track patterns, see file sizes, prune objects. Game dev and design friendly.

Fast. Actually.

Tauri + Rust backend. Native webview. Starts in under a second, not five.
