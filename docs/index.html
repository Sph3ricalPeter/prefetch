Open source · MIT licensed

Git client for people
who ship

GitKraken UX, Linear design, no subscription. Built with Tauri + Rust. 10MB, not 150.

Install

Windows
irm https://raw.githubusercontent.com/Sph3ricalPeter/prefetch/main/install.ps1 | iex
macOS
curl -fsSL https://raw.githubusercontent.com/Sph3ricalPeter/prefetch/main/install.sh | sh

What you get

Commit graph
Canvas-rendered, virtualized. Handles 100k+ commits.
Profiles
Auto-switch git identity by repo path. Work / personal, zero friction.
GitHub & GitLab
PR badges on branches. PAT in OS keychain. Click to open in browser.
Git LFS
Track patterns, see file sizes, prune objects. Game dev friendly.
Undo
Ctrl+Z backed by reflog. Works even for ops run in terminal.
10 MB
Tauri + Rust, not Electron. Native webview. Starts in under a second.
